# Urban Planning in Tier 2 Cities GitHub Repo
- A repo containing data specific to urban planning in tier 2 cities to build a machine learning pipeline.
  
## Purpose - Analysis of data for
- Better strategies for planning
- Mitigation for environmental issues
- Preventing ecosystem problems
- Decreasing temperatures and increasing AQI via greenery:infrastructure ratio

## Current Focus: Mysuru
- Mysuru is a swiftly developing Tier 2 city in the state of Karnataka
- Categorised into 65 wards
- Major Industries - Tourism & IT

## Area & Geographical Factors
- Average altitude = 770 metres
- Area = 286.05 km2
- 
- Located between 2 rivers - Kaveri & Kabini (tributary)
- Tropical savanna climate (Aw) bordering on a hot semi-arid climate (BSh) (Köppen climate classification)
- source - https://en.wikipedia.org/wiki/Mysore
